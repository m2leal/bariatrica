CIRURGIA BARIÁTRICA - REDUÇÃO DE ESTÔMAGO

CÁLCULO DA EVOLUÇÃO DE PERDA DE PESO

Entrada de Dados

Peso Inicial (Kg)
Peso Atual (Kg)
Altura (m) [ex. 1.84]
Data da Cirurgia
Excesso Inicial= (PI) - ((A2) x (25))
IMC Inicial= (PI) / (A2)
Excesso Atual= (PA) - ((A2) x (25))
IMC Atual= (PA) x (A2)
Excesso eliminado= ((PI - PA) * 100) / EI
Previsão 1º mês= (PI - (EI x 0.15)) - (PI - (EI x 0.20))
Previsão 2º mês= (PI - (EI x 0.25)) - (PI - (EI x 0.30))
Previsão 3º mês= (PI - (EI x 0.30)) - (PI - (EI x 0.40))
Previsão 6º mês= (PI - (EI x 0.45)) - (PI - (EI x 0.55))
Previsão 9º mês= (PI - (EI x 0.55)) - (PI - (EI x 0.65))
Previsão 12º mês= (PI - (EI x 0.65)) - (PI - (EI x 0.85))
A = Altura
EI = Excesso Inicial
PI = Peso Inicial
PA = Paso Atual

Saída de Dados

Excesso Inicial (Kg)
IMC* Inicial (Kg/m2)
Excesso Atual (Kg)
IMC Atual (Kg/m2)
Excesso eliminado (Kg - %)-
Cronograma de evolução prevista**
1º mês (15-20%)
2º mês (25-30%)
3º mês (30-40%)
6º mês (45-55%)
9º mês (55-65%)
12º ao 18º mês (65-85%)***
* Tabela referência de IMC disponível em http://www.centrobesidademetabolica.com.br/index.php/component/content/article/100.html
** Variações estimativas. Os resultados podem variar de pessoa para pessoa, de acordo com o metabolismo do paciente, nível de comprometimento pessoal, alimentação e prática de atividades físicas.
*** Ultrapassada esta margem de 85%, recomenda-se os resultados sejam avaliados por meio de equipamentos de alta precisão, denominados de bioimpedância corporal (InBody), que possibilita uma avaliação mais precisa da distribuição corporal da pessoa: composição líquida, estrutura óssea, massa muscular e percentual de gordura do paciente.
****DatePicker adaptado de http://www.nsftools.com/tips/DatePickerTest.htm
