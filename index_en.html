BARIATRIC SURGERY - STOMAC REDUCTION

EVOLUTION OF WEIGHT LOSS CALCULUS

Data Input

Inicial Weight (Kg)
Current Weight (Kg)
Height (m) [ex. 1.84]
Surgery Date
Inicial Excess= (PI) - ((A2) x (25))
Inicial BMI*= (PI) / (A2)
Current Excess= (PA) - ((A2) x (25))
Current BMI= (PA) x (A2)
Eliminated Excess= ((PI - PA) * 100) / EI
1º month forecast= (PI - (EI x 0.15)) - (PI - (EI x 0.20))
2º month forecast= (PI - (EI x 0.25)) - (PI - (EI x 0.30))
3º month forecast= (PI - (EI x 0.30)) - (PI - (EI x 0.40))
6º month forecast= (PI - (EI x 0.45)) - (PI - (EI x 0.55))
9º month forecast= (PI - (EI x 0.55)) - (PI - (EI x 0.65))
12º month forecast= (PI - (EI x 0.65)) - (PI - (EI x 0.85))
A = Altura
EI = Excesso Inicial
PI = Peso Inicial
PA = Paso Atual

Data Output

Inicial Excess (Kg)
Inicial BMI* (Kg/m2)
Current Excess (Kg)
Current BMI (Kg/m2)
Eliminated Excess (Kg - %)-
Evolution Schedule Forecast**
1º month (15-20%)
2º month (25-30%)
3º month (30-40%)
6º month (45-55%)
9º month (55-65%)
12º to 18º month (65-85%)***
* BMI reference table available at http://www.centrobesidademetabolica.com.br/index.php/component/content/article/100.html
** Estimated variations. The results may vary from person to person, according to the patient's metabolism, level of personal commitment, feeding and physical activity.
*** Exceeded the 85% margin, it is recommended that the results be evaluated by high-precision equipment, called body bioimpedance (InBody), which enables a more accurate assessment of the person's body distribution: liquid composition, bone structure, muscle mass and fat percentage of the patient.
****DatePicker adapted from http://www.nsftools.com/tips/DatePickerTest.htm
